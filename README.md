# Online Banking System – Java Console Project

This project is a console-based Online Banking System developed using Java. It allows users to view account details, deposit money, and withdraw funds with proper validation and error handling.

## Features
- View account details
- Deposit money
- Withdraw money with balance validation
- File-based data persistence
- Custom exception handling
- Menu-driven console interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File I/O and Serialization
- Exception Handling

## Project Structure
- Account.java – Stores account details and business logic
- AccountDAO.java – Handles file read/write operations
- OnlineBankingSystem.java – Main class for user interaction
- InsufficientFundsException.java – Custom exception class

## How to Run
1. Open the project in IntelliJ IDEA
2. Mark `src` as Sources Root
3. Run `OnlineBankingSystem.java`
